# CartPole Deep Q-Network (PyTorch)

This project implements a Deep Q-Network from scratch using PyTorch to solve the CartPole-v1 environment.  
It includes a replay buffer, target network updates, epsilon-greedy exploration, reward tracking, and a modular code structure.

## Project Structure
- `models/dqn.py` — Q-network
- `replay/replay_buffer.py` — Replay buffer implementation
- `utils.py` — Plotting utilities
- `config.py` — Hyperparameters
- `train.py` — Main training loop
